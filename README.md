# **RPG Game**


- Консольная игра RPG. | **(Практика Python)**

#### Что есть в игре:		 
- 4 противника
- 3 персонажа (с возможностью покупки и выбора персонажа)
- Профиль
- RPG режим
- Сохранение прогресса
